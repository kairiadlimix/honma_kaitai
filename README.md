# 本間解体工業様 向け提案プロジェクト

## プロジェクト概要

本間解体工業様の事業デジタル化・効率化を目的としたAI活用システム提案プロジェクトです。

### 提案の背景

解体業界における従来の業務プロセスの課題を解決し、AI技術を活用した業務効率化と現場管理の向上を目指します。

## 現場の課題

- **出来高支払い管理の課題**: 人間が行うため「感覚的」に管理され、緻密な管理が実現できていない
- **カメラデータの未活用**: 現場に設置しているカメラデータは、映像を目視するだけでデータとして活用しきれていない
- **見積作成のリスク**: 経験と知識が必要で、誤りや記載漏れが発生するリスクを抱えている

## 解決策：AI活用ソフトウェアシステム

### デモサイト
- **URL**: https://matlys-co-ltd.github.io/honma-demo/
- 本間解体工業様専用のシステムデモンストレーションサイト

### 主要機能

#### 1. 出来高請求管理機能
**概要**: 現場カメラデータを活用し、工程・項目毎に請求を管理し支払いの管理を行う

**導入効果**:
- 現場毎の健全なキャッシュフロー管理の実現 ← 黒字倒産リスクの回避
- 下請け企業への支払いを現場の状況に適した根拠ある金額と時期で支払い実現（脱感覚管理）
- カメラデータの利活用実現
- 現場データの蓄積

#### 2. 解体資材の回収プロセスAI自動化
**概要**: 資材の積載量をAIで計算、満載になることを確認し回収業者様への配車手配を行う

**費用対効果**:
- 1回あたりの手配・管理業務時間を1/3カット
- 未満載による回収車の待機時間カット
- 現場データの蓄積

#### 3. AIによる見積書のチェック・自動作成
**概要**: AIを活用したkintoneとの連携を実現

**機能詳細**:
- 過去見積データから作成した見積に金額ミスや項目漏れをAIが自動でチェック
- 将来的には見積書の自動作成が可能

**費用対効果**:
- 見積作成の社内標準化
- 見積ミスによる金銭的なトラブル、現場混乱の防止

## 本間解体工業様からの具体的な要望

### 重要度の高い機能
- **出来高管理**: 現場の進捗状況を正確に把握・管理
- **部係係数活用した見積を自動作成**: 職人の感覚に依存していた部分をシステム化
- **重機選定支援**: 工事条件から最適な重機を自動選定
  - 道路の広さ、転回できる広さ
  - クレーンはビルの上？地上？などの条件から判断

### 解体材管理の要望
- 解体材から年間の解体数量、金額を管理したい
- 発生材3,000t × 10万円/単価 などの計算
- マニフェスト（伝票が手書きの紙管理）⇨ デジタル化してt数を計算、KPIとしての指標

### 見積もり自動化の要望
- 大まかな見積もりを自動でエクセルにして作って欲しい
- 解体工事は10種類ほどの工事から成り立つ
- 直接仮設、仮仮設など10種類ほどの工事を有りなしで選択肢、現場情報を営業担当が入れると大まかなエクセルで抽出される
- BIMモデルの活用：外壁の足場の数、コンクリートの排出量、囲いの数などが図面から精度高く記載されている

## プロジェクトの進捗状況

- 各現場の請求ステータス・進捗管理を会社全体で共有したい
- ダッシュボードで権限分けして映し出す方法を提案
- 過去データから分析して欲しい（紙管理が懸念点）
- 最初の段階では進捗をカメラで見れるだけで十分

#### 社内の動き
- **解体原価の開発が2年かかり、まもなく完了**
- 完了次第、岩本さん・システム部門が独立予定
- 最終的には本間解体と直接契約して欲しい

#### 競合状況
- **燈（ひかり）**の動きあり
- サービス内でAIエージェント化したいという提案
- kintoneと連携してきた
- ただし、マトリスの開発力・提案力で本間社内ではマトリスの方が評価高い

#### 今後の動き
- **PJ推進役として岩本さん、マトリスが開発で取り組む**
- 本間解体として、ダッシュボードサービス等、他の解体・建築業者へ転用できるものを販売していこうという野望
- 弊社はアライアンス賛成
- ダッシュボード計画は副社長と岩本さんで進めていく
- 翌週、各部の部長を集めて請求管理の状況ヒアリングを予定
- 社内では実施に向け走る方向性

## 契約・運用に関する重要事項

### データ保管・管理
- **データ保管場所**: 本間解体工業様の既存システム（kintone等）を活用
- **データ所有権**: 本間解体工業様が所有、弊社は開発・運用支援
- **データセキュリティ**: 本間解体工業様のセキュリティポリシーに準拠
- **バックアップ**: 本間解体工業様の既存バックアップシステムを利用

### データ共有・活用（matlys向け）
- **匿名化データの提供**: 個人情報を除去した統計データの提供
- **API連携**: 本間解体工業様の承認のもと、matlysシステムへのデータ連携
- **プロダクト改善データ**: システム利用状況、エラー情報、パフォーマンスデータ
- **業界分析データ**: 解体業界全体のトレンド分析に活用可能な匿名化データ
- **機械学習モデル改善**: AI精度向上のための学習データ（匿名化済み）
- **データ提供契約**: 別途データ提供に関する契約を締結
- **データ利用目的**: プロダクト改善、新機能開発、業界分析のみ
- **データ保持期間**: 契約終了後も一定期間保持（匿名化済み）

#### 解体業界の匿名化データ具体例

**現場データ**
- 建物種別（住宅/商業施設/工場等）、構造（木造/RC造/鉄骨造等）
- 解体面積、階数、建築年数
- 解体工事期間、作業時間
- 重機使用状況、作業員数
- 天候、季節要因

**出来高・請求データ**
- 工程別進捗率、完了日数
- 請求金額（範囲指定）、支払いサイクル
- 出来高支払いのタイミング
- 現場別の収益性指標

**資材・廃棄物データ**
- 廃棄物種別、発生量（t数）
- 回収頻度、回収車両数
- 積載効率、待機時間
- リサイクル率、処理費用

**見積・契約データ**
- 見積項目、単価（範囲指定）
- 契約形態、工期
- 変更・追加工事の発生率
- 競合他社との比較データ

**システム利用データ**
- 機能別利用頻度、操作ログ
- エラー発生状況、パフォーマンス
- ユーザー行動パターン
- 改善要望、フィードバック

**AI分析データ**
- 画像認識の精度、誤認識パターン
- 予測モデルの性能指標
- 学習データの特徴量
- モデル改善の効果測定
